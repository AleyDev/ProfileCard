# Profile Card

Modern ve şık bir profil kartı uygulaması. React ve Tailwind CSS ile geliştirilmiştir.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Özellikler

- ✨ Modern ve responsive tasarım
- 🎨 Tailwind CSS ile stil
- 🖼️ Profil fotoğrafı
- 📊 İstatistikler (Projeler, Takipçiler, Takip)
- 🔗 Sosyal medya linkleri
- 💫 Hover efektleri ve animasyonlar
